プレビューデプロイを全て削除する。`[PREVIEW]` プレフィックス付きデプロイのみを対象とし、productionデプロイと `@HEAD`（開発用）は絶対に削除しない。

## 手順

### 1. デプロイメント一覧の取得

```bash
clasp deployments
```

### 2. プレビューデプロイの特定

デプロイメッセージに `[PREVIEW]` が含まれるものを全て抽出する。

見つからなければ「プレビューデプロイは見つかりませんでした。」と表示して処理を終了する。

### 3. 削除の確認

AskUserQuestionツールで確認する：
- 質問文: 「以下のプレビューデプロイを削除します:\n（該当デプロイのID・メッセージを列挙）\n本当に削除しますか？」
- 選択肢: ["全て削除する", "中止"]
- 「中止」→ 処理を終了

### 4. 削除の実行

各プレビューデプロイを順番に削除する：
```bash
clasp undeploy <deploymentId>
```

### 5. 結果の表示

削除完了後に以下を表示する：
- 削除したデプロイメントの数
- 削除したデプロイメントIDの一覧
